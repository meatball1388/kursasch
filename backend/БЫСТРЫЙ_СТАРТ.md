# БЫСТРЫЙ СТАРТ

## 1. Установи PostgreSQL
- Скачай: https://www.postgresql.org/download/windows/
- Установи, запомни пароль для `postgres`

## 2. Создай базу данных
- Открой **SQL Shell (psql)** из меню Пуск
- Нажимай Enter для всех вопросов
- Введи пароль postgres
- Выполни:
```sql
CREATE DATABASE creating_software;
\c creating_software;
```
- Открой файл `backend/create_tables.sql` и скопируй содержимое в SQL Shell

## 3. Настрой .env
- Открой `backend\.env`
- Замени `postgres123` на свой пароль:
```
DB_URL=postgresql://postgres:ТВОЙ_ПАРОЛЬ@localhost:5432/creating_software
```

## 4. Запусти бэкенд
- Открой `backend\start_server.bat`
- Должно появиться: `Uvicorn running on http://0.0.0.0:8000`

## 5. Запусти фронтенд
- Запусти XAMPP (Apache)
- Открой: http://localhost/курсач/

## Готово! 🎉

---

**Проверка:**
- http://localhost:8000/v0/version — должен вернуть версию API
- http://localhost/курсач/ — главная страница
